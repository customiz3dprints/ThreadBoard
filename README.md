# Thread Board

Thread Board is a fully client-side app to solve your own work like a detective. This was made for the HackClub event Flavortown.

## Access

Access it [here](https://threadboard-town.web.app). No login required.

## Features

- Make notes wherever your heart desires.
- Connect them with strings to link ideas and with solutions, or even sub-ideas (WIP)
- Unlimited canvas
- Import your boards, or export to share with friends or co-workers (WIP)
- Styling your notes and strings (WIP)

## AI usage

AI was used in the making of this project. This project wasn't "vibe-coded", rather used tools like ChatGPT to understand how to implement ideas.
