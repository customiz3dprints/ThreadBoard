# Thread Board

Thread Board is a fully client-side app to solve your own work like a detective. This was made for the HackClub event Flavortown.

## Access

Access it [here](https://threadboard-town.web.app). No login required. Or hell, you can download it, and host it yourself to add features you want, but isn't there.

## Features

- Make notes wherever your heart desires.
- Connect them with strings to link ideas and with solutions, or even sub-ideas by clicking the red circle buttons between two notes
- Unlimited canvas
- Import your boards, or export to share with friends or co-workers (WIP)
- Styling your notes and strings (WIP)
