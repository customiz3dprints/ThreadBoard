# Thread Board

Thread Board is a fully client-side app to solve your own work like a detective. This was made for the HackClub event Flavortown.

## Access

Access it [here](https://threadboard-town.web.app). No login required. Or hell, you can download it, and host it yourself to add features you want, but isn't there.

## Features

- Make notes wherever your heart desires.
- Connect them with strings to link ideas and with solutions, or even sub-ideas
- Unlimited canvas
- Import your boards, or export to share with friends or co-workers (WIP)
- Styling your notes (done) and strings (WIP)

## Usage
- drag notes with the dark part with LMB
- pan the whole canvas with MMB
- Use RMB to open the menu to: add notes, change their color, or delete them
- click (not drag) the circles of two notes to connect them
- click the title/content to change edit them
